# dBank Web App v1.2 Release Notes

**Release Date:** 2023-10-15

## Overview
Welcome to dBank Web App v1.2! This release focuses on improving account security, optimizing our transaction history dashboard, and introducing new features for international transfers.

## New Features
- **Enhanced Two-Factor Authentication (2FA):** We have introduced support for authenticator apps (e.g., Google Authenticator, Authy) in addition to SMS-based 2FA. Users can configure this in the Security Settings.
- **International Transfers Hub:** A new dedicated dashboard for initiating and tracking international SWIFT transfers. Real-time exchange rates are now displayed before confirming the transfer.
- **Dark Mode:** By popular demand, a system-wide Dark Mode is now available. Toggle this in your Profile Preferences.

## Improvements & Bug Fixes
- **Transaction Dashboard Load Times:** Optimized database queries have reduced the load time for the transaction history page by 40%.
- **Login Timeout Bug:** Fixed an issue where the session timeout warning was not appearing before automatically logging the user out.
- **Statement Generation:** Addressed a bug where generating a PDF statement for custom date ranges spanning more than 1 year would fail.

## Known Issues
- Push notifications for scheduled transfers may be delayed by up to 5 minutes. A hotfix is scheduled for next week.
- Users on older versions of Safari (v13 and below) might experience UI glitches on the International Transfers Hub. We recommend updating the browser.
